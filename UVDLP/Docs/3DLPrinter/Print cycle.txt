


 1° Command:  Sends to the printer the number of levels that make up the object . Use Byte A, B anc C (min 0 max 16777215)

            A  B  C
 BE 1D 00 00 05 00 E0	Pc -> Printer    

 B4 1D 00 00 05 00 D6 	Printer -> PC  
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 2° Command: Sends to the printer the number of steps of the motor, to indicate the thickness of the 1° layer. Use Byte B anc C (min 0 max 65535)

           A  B  C
 BE 15 00 01 40 01 14	Pc -> Printer    

 B4 15 00 01 40 01 0A 	Printer -> PC 
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

3° Command. Sends to the printer the number of steps of displacement of the axis z. At the end of printing a level, the z-axis moves upwards in the number of steps.
                      The printer start the projector and move the z-axis from home to the number of steps Offset.   Use Byte B anc C (min 0 max 65535)
 
            A  B  C
 BE 16 00 01 40 01 15	Pc -> Printer                                              
 
 B4 16 00 01 40 01 0B 	Printer -> PC 
 
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
4 ° Command: the z-axis stop , the printer sends the following command to the PC.
The software waits for this command before continuing.

 B4 20 00 00 00 00 D4
 
The software displays layer image 
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

5° Command :Sends to the printer, time of exposure of the layer. Use Byte B anc C (min 0 max 65535) 

            A  B  C
 BE 17 00 00 14 00 E9	Pc -> Printer     (14 Hex = 20 Dec = 2  Second)

 B4 17 00 00 14 00 DF 	Printer -> PC 

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

6° Command: the end of exposure  , the printer sends the following command to the PC.

 B4 21 00 00 00 00 D5 	Printer -> PC 

The software displays a black image
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

7° Command: Sends to the printer move bath with resin

 BE 18 00 00 00 00 D6	Pc -> Printer                               

 B4 18 00 00 00 00 CC	Printer -> PC  

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

8° Command: the end of bath move, the printer sends the following command to the PC.
 
 B4 22 00 00 00 00 D6  	Printer -> PC 

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
